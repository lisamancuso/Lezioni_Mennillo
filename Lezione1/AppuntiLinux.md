# Settare ambiente Linux per Git
1. Installare git sulla macchina con `sudo apt install git`
2. Settare username e password di github con i seguenti comandi
`git config --global user.name "tuoUsernameGitHub"`
`git config --global user.email"tuaEmailGitHub"`


# Comandi Git
- `git commit` Questo comando prepara i file in una sessione di staging
- `git push` Questo serve ad inviare il codice in remoto
- `git pull` Questo comando serve a recuperare i file da remoto
- `git merge` Questo comando serve a fondere i branches di un progetto
